> Convert hour to day or night: 8 → 'day'

Demo: [vovanr.github.io/hour-to-day-or-night][demo]

## Install

```shell
npm install --save hour-to-day-or-night
```

## Usage

```js
var hourToDayOrNight = require('hour-to-day-or-night');

hourToDayOrNight(8);
//=> 'day'

hourToDayOrNight(19);
//=> 'night'
```
